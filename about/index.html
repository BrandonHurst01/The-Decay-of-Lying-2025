The "EditionCrafter" view features a side by side visual of encoded and zoned IIIF images of the Akron MS and transcriptions and variants of the corresponding portion of the essay. Readers can switch between the five different encoded versions of the text from the dropdown menu at the top right of the viewer. These versions are:

Folio # Diplomatic Transcription: A Diplomatic Transcription of the Akron MS, which also includes in its display deleted or altered text present in the Berg Manuscripts housed at the New York Public Library.

Folio # Nineteenth Century: The Version Published in the periodical publication Nineteenth Century, with all MS, Editor, and Typesetter implemented alterations tracked.

Folio # Intentions 1891: The 1891 Intentions edition, which tracks changes introduced following its periodical publication in 1889.

Folio # Intentions 1894: The 1894 Intentions edition, mapping differences between the first and second edition of the collection.

Folio # Fluid: A Fluid Text Version that compiles all the alterations into a single version

Should users be interested in viewing or downloading the XML, which encodes all the changes according to the schema mapped below, selectign the "toggle xml" icon to the right of the file selection dropdown menu will display the XML for the selected file.

Using the navigation bar at the top of the screen, users can also select to view the "Reading Text" which includes a cleaned, compiled version of the "Fluid" files with accompanying commentary in the form of endnotes.

For ease of access and general information, the change encoding stated in TEI header information is included here:

  • change xml:id="StA": Stage A: Early drafts
  • change xml:id="StAb": Stage Ab: Variants extant in incomplete holograph held in the New York Public Library's Berg Collection
  • change xml:id="StB": Stage B: Faircopy manuscript delivered to The Nineteenth Century. Herman Meuhlstein Collection, University of Akron.
  • change xml:id="StBa": Stage Ba: Ink revisions, in Wilde's hand, in the Akron Manuscript
  • change xml:id="StBb": Stage Bb: Pencil revisions in Wilde's hand
  • change xml:id="StC": Stage C: Corrections and changes to Stage B text by the compositors or editors of The Nineteenth Century
  • change xml:id="StD": Stage D: Revisions made between NC and the 1891 Intentions
  • change xml:id="StE": Stage E: Revisions introduced between the 1891 and 1894 editions of Intentions

There are a select few Stage C changes which have not been assigned a "hand" attribute. These are changes in diction and phrasing that are not present as revisions to the MS but have been implemented in the 1889 published text.
